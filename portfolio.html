Project Name:Employee and Department Management System

Description:

This project is a web-based application designed to manage employees and departments within an organization. It provides an intuitive interface for administrators to perform CRUD operations (Create, Read, Update, Delete) on employee and department records. The system allows users to add, modify, delete, and view employee details, including their CIN (National Identity Card number), name, surname, grade, and assigned department. Additionally, it enables the management of departments, ensuring a structured and organized workflow. The application is built with a focus on simplicity, efficiency, and user-friendliness, making it suitable for small to medium-sized organizations. It ensures data integrity and security while providing a seamless experience for managing human resources.

Key FeaturesEmployee Management:

Add new employees with details such as CIN, name, surname, grade, and department.
Modify existing employee information.
Delete employee records.
View a list of all employees with their details.
Department Management:
Add new departments.
Modify department names.
Delete departments.
View a list of all departments.
User-Friendly Interface:
Simple and intuitive design for easy navigation.
Responsive layout for use on different devices.
Data Integrity:
Ensures that employees are linked to valid departments.
Prevents orphaned records (e.g., employees without a department).

Technologies Used:

Front-end:
HTML: For structuring the web pages.
CSS: For styling and designing the user interface.
Back-end:
PHP: For server-side logic and handling database operations.
Database:
MySQL: For storing and managing employee and department data.
Development Environment:
XAMPP: A local server environment for testing and development (includes Apache and MySQL).
Tools:
Visual Studio Code: A code editor for writing and debugging the application.
phpMyAdmin: For managing the MySQL database.

How It Works

1- Employee Management:
* Administrators can add new employees by filling out a form with their details.
* Existing employee records can be updated or deleted as needed.
* A table displays all employees, including their assigned department.
2- Department Management:
* Administrators can add, modify, or delete departments.
* Departments are linked to employees, ensuring a clear organizational structure.
3- Database Interaction:
* PHP scripts handle communication between the front-end and the MySQL database.
* SQL queries are used to perform CRUD operations on the database.
